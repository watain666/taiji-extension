# TâiJī Chhân 台字田

A browser extension for searching Taiwanese words and characters on [ji.taioan.org](https://ji.taioan.org).

## Development

This extension uses pure JavaScript with no build tools required.

1. Make changes to any file (`manifest.json`, `background.js`, etc.)
2. Open your browser's extension management page (`chrome://extensions` or `edge://extensions`)
3. Click the **Reload** button for this extension to apply changes

## Publishing

To publish, simply zip the entire directory:

```bash
zip -r extension.zip .
```

Then upload the `.zip` file to the Chrome Web Store.
